# JogoGourmet

## Descrição
Essa aplicação é um jogo de adivinhação de pratos, onde o usuário deve responder as perguntas do jogo e no final o jogo deve acertar o prato que o usuário pensou.

## Requisitos

Você deve ter o .NET Core 6.0 instalado na sua máquina para executar a aplicação.
https://dotnet.microsoft.com/pt-br/download/dotnet/6.0
## Tecnologias
- .NET Core 6.0
- C#

## Como executar
Para executar a aplicação, basta executar o comando abaixo na pasta raiz do projeto:
```
dotnet run
```
